## 目录

这是一个通过 CSS 实现的 3D 跳跃文字加载器效果。它利用 CSS 的动画和变换属性，创建了一个立体感强烈的文字动画效果。

### 核心实现原理

- **HTML 结构**：使用 `<h1>` 标签包裹多个 `<span>` 元素，每个 `<span>` 元素代表一个字母，并使用 `style="--i:n"` 设置动画的延迟。
- **CSS 核心实现**：
  - **居中与背景**：`body` 使用 `min-height: 100vh` + Flex 居中，深色背景 `#ff5757` 突出发光效果。
  - **排列方式**：`h1 { display: flex; }` 横向排列各字母。
  - **基础字体与样式**：`font-size: 5em`、`letter-spacing: 15px`、初始文字颜色 `#262626`（暗灰）。
  - **动画设定**：`animation: animate 0.5s linear infinite;` 并通过 `animation-delay: calc(var(--i)*0.2s)` 为每个字母增加 0.2s 递增延迟，形成由左到右的跳跃效果。

### 关键帧 `@keyframes animate`

- **0% ~ 90%**：文字保持暗灰色，`text-shadow: none`，处于“未点亮”状态。
- **100%**：文字变为亮黄色 `#fff`，并叠加两层发光阴影：
  - `0 0 7px #fff`（近处黄光）
  - `0 0 50px #ff6c00`（远处橙光）
    结合线性无限循环与递增延迟，显示为持续向前流动的“跳跃文字”加载效果。

### 视觉效果要点

- 深色背景下的高亮文本光晕更显著。
- 通过自定义属性配合 `calc()` 统一控制延迟，结构简洁、扩展性好（新增字母仅需递增 `--i`）。

### 技术亮点

- **CSS 动画**：仅使用 CSS 动画实现动画效果，避免触发回流和重绘，保证流畅性。
- **响应式设计**：固定尺寸设计，适配各种屏幕。
- **性能优化**：使用 CSS 动画而非 JavaScript，确保流畅性能。

整体上，该效果以少量 DOM 与纯 CSS 构建出具有立体感的跳跃文字动画，层次分明且具备优秀的渲染性能。
