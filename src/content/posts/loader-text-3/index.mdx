## 目录

## CSS 动画亮点

- 通过 `ul` 设置 `display: flex`，将每个字母的 `li` 横向排列，形成连贯的 Loading 文本。
- `animation: animate 3s linear infinite` 让文本不断从左至右滑动，配合透明度与位移变化制造流动感。
- `letter-spacing` 在关键帧间由 `20px` 压缩到 `-20px`，营造文字聚拢后再散开的节奏变化。
- 颜色在动画末尾从 `#f00` 渐变到 `#00f`，带来由红到蓝的视觉转换。

## 关键知识点

- 利用 `calc(var(--i) * 0.2s)` 为每个 `li` 设置不同的 `animation-delay`，实现级联错峰动画。
- `transform: translateX()` 配合透明度控制，负责文本的进场与离场，使动画首尾衔接自然。
- 通过 `min-height: 100vh` 与 `justify-content`/`align-items` 将场景垂直水平居中，保证在不同视口下的位置一致。
