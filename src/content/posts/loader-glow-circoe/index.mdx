## CSS 效果与关键知识点

### 视觉效果

这是一个**发光圆环加载器**，具有以下视觉效果：

1. **静态圆环**：深灰色边框的圆形容器，中心显示 "Loading" 文字
2. **旋转边框**：黄色边框在圆环顶部和右侧旋转，形成动态加载效果
3. **旋转指针**：黄色圆点指针从中心向右延伸，围绕中心旋转
4. **发光效果**：文字和指针都有黄色发光阴影，增强视觉层次

### 关键 CSS 知识点

#### 1. **伪元素 ::before 创建旋转边框** (30-42 行)

```css
.loader::before {
  content: "";
  position: absolute;
  top: -3px; /* 边框宽度 */
  left: -3px; /* 边框宽度 */
  width: 100%;
  height: 100%;
  border-radius: 50%;
  border: 3px solid transparent;
  border-top: 3px solid #fff000;
  border-right: 3px solid #fff000;
  animation: animateCircle 2s linear infinite;
}
```

- **作用**：使用伪元素创建独立的旋转边框层
- **定位技巧**：`top: -3px; left: -3px` 偏移边框宽度，使伪元素边框与主元素边框重叠
- **边框技巧**：使用 `transparent` 作为基础边框，只显示 `border-top` 和 `border-right`
- **动画**：`animateCircle` 实现 360° 无限旋转

#### 2. **transform-origin 控制旋转中心** (51 行)

```css
.loader span {
  transform-origin: left;
  animation: animate 2s linear infinite;
}
```

- **作用**：设置旋转中心点为元素的左边缘
- **效果**：指针从中心点（50%）向右延伸，旋转时围绕左端点旋转
- **关键**：配合 `left: 50%` 实现指针从圆心旋转的效果

#### 3. **calc() 精确居中定位** (46 行)

```css
top: calc(50% - 2px);
```

- **作用**：精确计算垂直居中位置
- **原理**：50% 减去元素高度的一半（4px / 2 = 2px），实现完美居中
- **优势**：比使用 `transform: translateY(-50%)` 更直接，避免额外的变换

#### 4. **双重旋转动画**

**指针旋转动画** (65-72 行)

```css
@keyframes animate {
  0% {
    transform: rotate(45deg);
  }
  100% {
    transform: rotate(405deg);
  }
}
```

- **起始角度**：45deg（指向右上角）
- **旋转角度**：360°（405deg - 45deg = 360deg）
- **效果**：指针完整旋转一周

**边框旋转动画** (73-80 行)

```css
@keyframes animateCircle {
  0% {
    transform: rotate(0deg);
  }
  100% {
    transform: rotate(360deg);
  }
}
```

- **效果**：边框从 0° 旋转到 360°
- **同步**：与指针动画时长相同（2s），但起始角度不同，形成错位效果

#### 5. **发光效果实现** (26, 28, 63 行)

```css
/* 容器阴影 */
box-shadow: 0 0 20px rgba(0, 0, 0, 0.5);

/* 文字发光 */
text-shadow: 0 0 10px #fff000;

/* 指针发光 */
box-shadow: 0 0 20px #fff000;
```

- **box-shadow**：创建外发光效果，`0 0` 表示无偏移，只产生模糊
- **text-shadow**：为文字添加发光效果
- **颜色统一**：使用 `#fff000`（黄色）保持视觉一致性

#### 6. **绝对定位布局** (15, 32, 45 行)

```css
.loader {
  position: relative; /* 创建定位上下文 */
}
.loader::before {
  position: absolute; /* 相对于 .loader 定位 */
}
.loader span {
  position: absolute; /* 相对于 .loader 定位 */
}
```

- **定位层级**：`relative` 父元素 + `absolute` 子元素
- **优势**：精确控制伪元素和指针的位置，不受文档流影响

#### 7. **Flexbox 居中布局** (7-9, 22-24 行)

```css
body {
  display: flex;
  justify-content: center;
  align-items: center;
}
.loader {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

- **外层**：页面水平垂直居中加载器
- **内层**：加载器内部文字居中显示

### 技术亮点总结

✅ **伪元素技巧**：使用 `::before` 创建独立的动画层，避免影响主元素  
✅ **定位精确**：通过 `calc()` 和负值偏移实现像素级精确对齐  
✅ **旋转中心控制**：`transform-origin` 实现从特定点旋转的效果  
✅ **双重动画**：边框和指针独立旋转，形成丰富的视觉层次  
✅ **发光效果**：`box-shadow` 和 `text-shadow` 营造科技感  
✅ **性能优化**：使用 `transform` 和 `opacity` 实现硬件加速动画
