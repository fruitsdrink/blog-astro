本示例呈现"四点矩形巡航"的加载效果：四个彩色光点（青、绿、红、黄）沿着矩形的四个角点循环移动，形成连续的巡航动画。每个点通过 `box-shadow` 营造霓虹发光效果，使用负延迟（`animation-delay: -0.5s, -1s, -1.5s`）实现错峰移动，让四个点在不同时间点出发，形成流畅的追逐效果。

**关键知识点：**

1. **CSS 动画关键帧（@keyframes）**：定义矩形路径动画，从左上角（0%, 0%）→ 右上角（50%, 0%）→ 右下角（50%, 50%）→ 左下角（0%, 50%）→ 回到左上角，形成完整的矩形循环路径。

2. **负延迟错峰效果（Negative Animation Delay）**：通过 `animation-delay: -0.5s, -1s, -1.5s` 让后三个点分别提前 0.5s、1s、1.5s 开始动画，实现四个点错峰移动，避免同时出发造成的视觉重叠。

3. **霓虹发光效果（Box Shadow）**：每个点使用双层 `box-shadow`（`0 0 10px` 和 `0 0 20px`）配合不同颜色，营造霓虹灯般的发光效果，增强视觉吸引力。

4. **绝对定位控制**：使用 `position: absolute` 配合 `left` 和 `top` 属性，通过动画改变位置值，实现点的移动效果。

5. **动画暂停点设计**：在关键帧的 5%、25%、55%、80% 处设置相同的坐标值，让点在每个角点短暂停留，形成"停顿-移动-停顿"的节奏感。
