## 代码解析

### HTML 结构

- **`<ul>` + 多个 `<li>`**: 由一个无序列表构成，分别渲染字母 L O A D I N G。
- **`style="--i:n"`**: 每个 `<li>` 内联设置 CSS 自定义属性 `--i`，用于控制动画的延迟，实现“逐字亮起”的流水灯效果。

### CSS 核心实现

- **居中与背景**: `body` 使用 `min-height: 100vh` + Flex 居中，深色背景 `#262626` 突出发光效果。
- **排列方式**: `ul { display: flex; }` 横向排列各字母。
- **基础字体与样式**: `font-size: 5em`、`letter-spacing: 15px`、初始文字颜色 `#484848`（暗灰）。
- **动画设定**: `animation: animate 1.4s linear infinite;` 并通过 `animation-delay: calc(0.2s * var(--i))` 为每个字母增加 0.2s 递增延迟，形成由左到右的亮起波。

### 关键帧 `@keyframes animate`

- **0% ~ 90%**: 文字保持暗灰色，`text-shadow: none`，处于“未点亮”状态。
- **100%**: 文字变为亮黄色 `#fff900`，并叠加两层发光阴影：
  - `0 0 7px #fff900`（近处黄光）
  - `0 0 50px #ff6c00`（远处橙光）
  结合线性无限循环与递增延迟，显示为持续向前流动的“闪烁发光文字”加载效果。

### 视觉效果要点

- 深色背景下的高亮文本光晕更显著。
- 通过自定义属性配合 `calc()` 统一控制延迟，结构简洁、扩展性好（新增字母仅需递增 `--i`）。
