## 简单文字逐字跳动加载器效果分析

这是一个由 "LOADING" 文字组成的逐字跳动加载动画。每个字母依次高亮、上跳并右移，通过延迟错峰形成波浪式的视觉节奏。

### 结构与布局
- 容器 `.loader` 包含 7 个 `<span>` 元素，分别展示 "LOADING" 的每个字母
- 每个 `<span>` 通过内联样式 `style="--i: n"` 设置 CSS 变量，用于计算动画延迟
- 文字大小 `font-size: 24px`，初始颜色为半透明白色 `rgba(255, 255, 255, 0.1)`

### 视觉设计
- **初始状态**：字母呈现低透明度白色，营造“待激活”的视觉状态
- **激活状态**（25% 关键帧）：
  - 颜色变为黄色 `yellow`，形成强烈对比
  - 向上移动 `translateY(-15px)` 产生跳跃感
  - 水平右移 `margin-left: 10px` 增加动感
  - 添加文字阴影 `text-shadow: 0 15px 5px rgba(0, 0, 0, 1)` 增强立体感与深度
- **恢复状态**：颜色与位置回到初始透明状态，完成一个循环

### 动画与节奏
- 核心动画 `@keyframes animate`：0% → 25% → 100% 的三阶段序列
  - 0% 与 100% 保持一致的初始状态，25% 为峰值高亮
  - 动画时长 2s，`ease` 缓动（默认），无限循环
- **错峰延迟**：通过 `animation-delay: calc(var(--i) * 0.1s)` 实现
  - 每个字母延迟 `0.1s` 的倍数（第1个字母 0.1s，第2个 0.2s...），形成波浪式的逐字激活序列
  - 这种延迟让整体动画更具流动感，避免所有字母同步跳动的呆板

### 技术实现细节
- **CSS 变量应用**：使用 `--i` 自定义属性与 `calc()` 函数动态计算延迟，代码简洁且易于扩展
- **过渡效果**：`transition: all 0.5s` 为动画提供平滑的过渡，但在关键帧动画中作用有限
- **位移方式**：垂直位移使用 `transform: translateY()`（性能优），但水平位移使用 `margin-left`（会触发重排，性能较差）

### 性能考虑
- ✅ 垂直位移使用 `transform`，由合成层驱动，性能良好
- ⚠️ 水平位移使用 `margin-left` 会触发布局回流，建议改为 `transform: translateX()` 以提升性能
- 颜色与阴影变化会触发重绘，但影响相对较小

### 适用场景
- 需要文字提示的加载界面
- 轻量级、低复杂度的加载指示器
- 强调文字内容的等待状态

总体而言，该效果以极简的 DOM 结构与纯 CSS 实现了清晰的逐字波浪动画，视觉节奏感强。如将 `margin-left` 替换为 `transform`，可进一步优化性能。
