## 效果介绍

**Loader-rainbow** 是一个充满活力的彩虹圆环加载器，通过多层半圆环的旋转动画创造出绚丽多彩的视觉效果。

### 核心实现原理

- **彩虹配色**：使用 7 种颜色（红、橙、黄、绿、蓝、靛、紫）模拟彩虹光谱
- **半圆环设计**：通过设置 `border-bottom-color` 和 `border-left-color` 为透明，创造半圆形效果
- **分层叠加**：7 个不同大小的圆环从内到外依次排列，形成层次感
- **旋转动画**：每个圆环从 -45° 旋转到 315°，创造出波浪般的滚动效果
- **延迟动画**：每个圆环延迟 0.2 秒启动，形成连续波浪效果

### 视觉效果特点

- **彩虹渐变**：红 → 橙 → 黄 → 绿 → 蓝 → 靛 → 紫的完整彩虹光谱
- **波浪滚动**：圆环依次旋转，形成流畅的波浪滚动动画
- **深度层次**：从 60px 到 240px 的尺寸渐变，营造深度感
- **阴影效果**：`box-shadow` 增加立体感和层次

### 技术亮点

- **边框技巧**：巧妙利用透明边框创造半圆形
- **动画延迟**：使用 `animation-delay` 创造连续动画效果
- **旋转变换**：`rotate(-45deg)` 到 `rotate(315deg)` 的 360° 旋转
- **绝对定位**：使用绝对定位和 `translate` 精确控制位置
- **无限循环**：`infinite alternate` 实现往返无限循环

### 关键技术细节

#### 1. 半圆环实现

```css
border: 15px solid #000;
border-bottom-color: transparent !important;
border-left-color: transparent !important;
```

通过将底部和左侧边框设为透明，创造上半圆环效果。

#### 2. 彩虹配色

- 红色 `#FF0000`：最内层（60px）
- 橙色 `#FF7F00`：第二层（90px）
- 黄色 `#FFFF00`：第三层（120px）
- 绿色 `#00FF00`：第四层（150px）
- 蓝色 `#0000FF`：第五层（180px）
- 靛色 `#4B0082`：第六层（210px）
- 紫色 `#9400D3`：最外层（240px）

#### 3. 动画设置

- **动画时长**：5 秒完成一个周期
- **动画模式**：`alternate` 实现往返动画
- **旋转角度**：从 -45° 到 315°，覆盖 360° 旋转
- **延迟时间**：每个圆环递增 0.2 秒延迟

#### 4. 定位技巧

```css
top: 100%;
left: 50%;
transform: translate(-50%, -50%);
```

使用 `top: 100%` 初始位置在底部，然后通过 `translate` 居中，配合动画实现从底部弹出的效果。

### 应用场景

这种效果常用于：

- 游戏界面加载
- 娱乐类网站
- 需要突出活力和趣味性的场景
- 儿童主题应用
- 创意展示页面

这种加载器通过彩虹色和流畅动画，能够有效吸引用户注意力，传达"正在加载"的视觉信息，同时带来愉悦的视觉体验。
