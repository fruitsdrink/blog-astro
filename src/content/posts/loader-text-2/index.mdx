## 目录

本示例呈现"文字加载动画"效果：通过多个方块容器，每个容器内显示一个字母（L、O、A、D、I、N、G），奇偶元素分别从上下两个方向滑入，配合缩放动画，创造出流畅的文字加载效果。

## 技术关键点

1. **伪元素内容生成**：使用 `::before` 伪元素配合 `content: attr(data-attr)` 属性，从 HTML 的 `data-attr` 属性中读取并显示文本内容。这种方式避免了在 HTML 中重复书写文本，实现了内容与样式的分离：
   ```css
   ul li:nth-child(even):before {
     content: attr(data-attr);
   }
   ```

2. **CSS 自定义属性延迟**：通过 CSS 变量 `--i` 和 `calc()` 函数实现动画延迟的递增效果，让每个字母的动画错开时间，形成波浪式的动画效果：
   ```css
   animation-delay: calc(var(--i) * 0.1s);
   ```
   每个元素根据其 `--i` 值（0, 1, 2, 3...）计算不同的延迟时间，创造出连续的动画序列。

3. **奇偶元素差异化动画**：
   - **偶数元素（even）**：从上方滑入（`top: -100%` → `top: 0` → `top: 100%`），配合缩放从大到小（`scale(1)` → `scale(0.5)` → `scale(0.2)`）
   - **奇数元素（odd）**：从下方滑入（`top: 100%` → `top: 0` → `top: -100%`），同样配合缩放动画
   这种交替的动画方向创造出视觉上的节奏感和层次感。

4. **关键帧动画设计**：
   - `0%`：元素在容器外（上方或下方），完整大小（`scale(1)`）
   - `30%`：元素滑入容器中心位置（`top: 0`），缩小到一半（`scale(0.5)`）
   - `70%`：元素保持在中心位置，维持一半大小
   - `100%`：元素滑出容器（相反方向），缩小到最小（`scale(0.2)`）
   这种设计让每个字母在中心位置停留一段时间，确保文字清晰可见。

5. **绝对定位与层叠**：使用 `position: absolute` 让伪元素脱离文档流，覆盖在容器上方。通过设置 `width: 100%` 和 `height: 100%` 以及 `line-height: 150px`（与容器高度相同），实现文字在容器中的垂直居中显示。

6. **视觉增强效果**：
   - `box-shadow: 0 0 15px rgba(0,0,0,1)` 为每个方块和文字添加阴影，增强立体感
   - 黄色背景（`background: #ff0`）与深色容器背景形成强烈对比，突出文字内容
   - 深色背景（`#1d1d1d` 和 `#262626`）营造出现代感的暗色主题

7. **动画参数设置**：
   - `1.5s` 的动画时长确保动画流畅且不会过快
   - `linear` 缓动函数让动画保持匀速，与延迟时间配合形成规律的节奏
   - `infinite` 实现无限循环，适合加载场景的持续展示

这种设计通过简单的 CSS 技术组合，创造出富有动感的文字加载动画，每个字母的独立动画和整体协调的节奏感，让加载过程变得生动有趣。
