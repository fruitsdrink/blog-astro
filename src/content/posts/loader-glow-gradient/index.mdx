## 效果介绍

**Loader-glow-gradient** 是一个极具视觉冲击力的发光渐变圆环加载器，它通过巧妙的 CSS 技术实现了霓虹灯般的炫酷效果。

### 核心实现原理

- **渐变色彩**：使用 `linear-gradient(#14ffe9, #ffeb3b, #ff00e0)` 创建青绿-黄色-粉色的三色渐变
- **多层模糊**：通过 4 个 `span` 元素叠加，分别应用 5px、10px、25px、50px 的模糊滤镜
- **旋转动画**：主容器以 0.5 秒的周期进行 360° 无限旋转
- **内圆遮罩**：使用 `::after` 伪元素创建内圆，露出外圈的发光效果

### 视觉效果特点

- **霓虹发光**：多层模糊叠加创造出真实的霓虹灯发光效果
- **色彩丰富**：青绿、黄色、粉色的渐变组合营造出科技感
- **动态旋转**：平滑的旋转动画增强视觉吸引力
- **层次分明**：不同模糊程度的层叠创造出深度感

### 技术亮点

- **滤镜叠加**：巧妙利用 `filter: blur()` 的多层叠加效果
- **伪元素遮罩**：通过 `::after` 创建内圆遮罩，突出外圈发光
- **性能优化**：使用 CSS 动画而非 JavaScript，确保流畅性能
- **响应式设计**：固定尺寸设计，适配各种屏幕
