## 双重发光加载环效果分析

这是一个由两层同心圆环组成的发光加载动画。外层与内层分别以相反方向旋转，并在环上设置发光“指示点”，形成富有节奏的双环追逐效果。

### 结构与布局

- 容器 `.loader` 固定尺寸为 200×200，作为两层圆环的包裹框
- 每一层使用 `.face` 充满容器，并通过 `border-radius: 50%` 构成圆形
- 内层 `.face2` 相比外层 `.face1` 向内收缩 30px，形成双环层级

### 视觉设计与发光

- 通过仅在外环设置 `border-left` 与 `border-top`（黄色），内环设置 `border-right` 与 `border-bottom`（蓝色），制造“半环”视觉，从而突出旋转方向
- 暗色背景 `#060606` 与圆面背景 `#090909`、阴影 `box-shadow` 提升对比度与立体感
- 发光点由 `.circle::before` 实现：小圆点叠加多层 `box-shadow`，分别使用黄色 `#ffff00` 与蓝色 `#03a9f4`，呈现强烈辉光

### 动画与节奏

- 核心动画 `@keyframes animate`：0°→360° 的连续旋转，时长 3s，线性无限循环
- 外环 `.face1` 正向旋转；内环 `.face2` 使用 `animation: ... reverse` 反向旋转，形成双向对流的动感
- `.circle` 位于环的半径处，`transform-origin: left` 并以 `top: calc(50% - 1px)` 对齐圆心，以 `rotate(-45deg)` 设定初始相位，使发光点从对角位置起步，增强视觉层次

### 性能与可维护性

- 动画仅依赖 `transform` 与 `box-shadow`，大部分由合成层驱动，性能较优
- 颜色、尺寸、收缩距离（30px）等参数清晰独立，便于换肤与扩展（如增加第三层环）

### 适用场景

- 需要科技感、对比鲜明的加载指示
- 双向旋转适合强调“处理进行中”的状态反馈

总体而言，该效果以极简的 DOM 结构与纯 CSS 实现了层次分明的双环发光旋转，方向对冲与辉光点同步运动带来强烈的节奏与科技质感。
