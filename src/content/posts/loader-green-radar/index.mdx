## 绿色雷达扫描效果分析

这是一个模拟雷达扫描界面的加载动画。以世界地图为背景，包含网格线、距离环、旋转扫描扇面以及闪烁的目标指示点，形成完整的雷达系统视觉效果。

### 结构与布局

- 容器 `.radar` 为 480×480 的圆形（`border-radius: 50%`），白色边框 20px，作为雷达屏幕主体
- 背景采用双层：底层黑色 `#000`，上层叠加世界地图图片（`background-size: cover`），营造真实雷达屏幕质感
- 8 个 `<li>` 元素分工明确：前 4 个用于网格线，第 5-7 个用于距离环，第 8 个用于扫描扇面

### 视觉设计与层次

- **网格系统**：前 4 个 `<li>` 作为线条，位于容器中心（`top: 50%`），通过 `transform: rotate()` 分别旋转 0°、90°、45°、-45°，形成十字线和对角线网格
- **距离环**：第 5-7 个 `<li>` 为同心圆边框，尺寸分别为 120px、240px、360px，通过 `translate(-50%, -50%)` 居中，使用绿色边框 `rgba(0,255,0,1)` 模拟距离刻度
- **发光效果**：容器使用双重 `box-shadow`：外部阴影 `0 8px 0 #c5c5c5` 营造立体感，内部阴影 `inset 0 0 100px rgba(0,255,0,0.5)` 产生绿色内发光，增强雷达屏幕氛围
- **目标指示点**：通过 `::before` 和 `::after` 伪元素创建两个绿色圆点（10×10px），分别位于地图上的不同位置（约 55%/80% 和 45%/70%），使用 `filter: blur(2px)` 产生柔和的发光效果

### 动画与节奏

- **扫描扇面动画** `@keyframes animate`：第 8 个 `<li>` 使用 `linear-gradient(45deg, #00ff00 0%, transparent 50%)` 创建从左上角向右下角渐变的扇面，通过 `transform-origin: top left` 将旋转原点设在左上角，执行 0°→360° 的连续旋转，时长 2s，线性无限循环，模拟雷达扫描线持续转动
- **目标闪烁动画** `@keyframes glow`：两个目标点使用透明度闪烁，从 0→1→0，时长 1s，线性无限循环，模拟雷达发现目标时的闪烁提示
- **注意**：`glow` 动画定义中存在重复的 `0%` 关键帧（第 50 行与第 56 行），虽不影响效果，但建议修复为标准的 `0% → 50% → 100%` 结构

### 性能与可维护性

- 动画主要依赖 `transform` 旋转，由合成层驱动，性能优异
- 使用伪元素减少 DOM 节点，结构简洁高效
- 地图背景和发光效果使用图片与 `box-shadow`，视觉层次丰富但需注意资源加载

### 适用场景

- 需要科技感、军事风格的加载界面
- 数据监控、定位追踪类应用的状态指示
- 游戏或可视化项目的雷达系统 UI

总体而言，该效果以简洁的 DOM 结构与纯 CSS 实现了完整的雷达扫描视觉系统，地图背景、网格环线、旋转扫描与目标点闪烁的协同，营造出沉浸式的雷达界面体验。
