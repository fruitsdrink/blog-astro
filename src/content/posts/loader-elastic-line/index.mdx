## 效果介绍

**Loader-elastic-line** 是一个优雅流畅的弹性线条加载器，通过 SVG 路径动画和模糊滤镜创造出丝滑的线条变形效果。

### 核心实现原理

- **SVG 路径动画**：使用 `d: path()` 属性实现路径变形，线条从一种曲线形态过渡到另一种形态
- **双层叠加**：创建两个相同的 SVG 元素，第二层应用 40px 的模糊滤镜营造发光效果
- **颜色渐变**：从粉色 `#ff0092` 过渡到绿色 `#00ff00`，增强视觉冲击
- **弹性动画**：使用 `ease-in-out` 缓动函数，让动画更加自然流畅

### 视觉效果特点

- **流畅变形**：线条在不同曲线形态间平滑过渡，营造出橡皮筋般的弹性感
- **发光效果**：模糊滤镜创造出的光晕效果让线条更加突出
- **色彩变化**：从粉色到绿色的渐变更替增强动态感
- **循环播放**：2 秒周期无限循环，保持持续的动作感

### 技术亮点

- **SVG 路径控制**：通过动态改变 `d` 属性实现复杂的曲线变形
- **滤镜特效**：利用 `filter: blur()` 创造发光效果
- **CSS 动画**：纯 CSS 实现，性能优异，无需 JavaScript
- **stroke 属性**：通过改变 stroke 颜色实现色彩渐变

### 关键技术细节

- **路径定义**：使用三次贝塞尔曲线 `C` 定义平滑的曲线路径
- **动画关键帧**：在 0% 和 50% 定义两种不同的曲线形态，实现往返动画
- **模糊深度**：40px 的模糊程度恰到好处，既突出发光又不失清晰度
- **定位调整**：使用 `translate(50px, 50%)` 微调位置，确保视觉居中

这种效果常用于现代化网站、数据可视化界面或需要突出优雅感的场景，能够有效吸引用户注意力并传达"正在处理"的视觉信息。
