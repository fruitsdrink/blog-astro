## 3D 嵌套方形加载器效果分析

这是一个由三层嵌套方形组成的 3D 旋转加载器。每层以不同速度围绕 Y 轴旋转，配合黄色高光装饰，形成具有深度层次的视觉追逐效果。

### 结构与布局
- 外层容器 `.loader` 设置 `perspective: 600px`，创建 3D 透视空间，尺寸固定为 150×150
- 三层 `<span>` 元素通过 `position: absolute` 与 `inset` 属性实现嵌套：
  - 第一层 `inset: 0`（外层，150×150）
  - 第二层 `inset: 20px`（中层，110×110）
  - 第三层 `inset: 40px`（内层，70×70）
- 每个 `span` 使用 `border: 10px solid #262626` 与 `border-radius: 4px`，构成圆角方形边框框架

### 视觉设计与高光
- 深色背景 `#000` 与深灰边框 `#262626` 提供低对比度基础，突出旋转动感
- 每层通过伪元素 `::before` 与 `::after` 添加黄色 `#ff0` 高光装饰：
  - `::before`：位于左上角（`top: -10px; left: -10px`），高度为 50%，营造顶部左侧高光
  - `::after`：位于右下角（`bottom: -10px; right: -10px`），高度为 50%，营造底部右侧高光
- 高光区域配合 `border-top-left-radius` 与 `border-bottom-right-radius`，保持与主边框的圆角一致性
- 这种对角高光设计强化了 3D 旋转时的明暗变化，增强立体感

### 动画与节奏
- 核心动画 `@keyframes animate`：围绕 Y 轴做 0°→360° 连续旋转，线性无限循环
- 三层旋转速度差异显著：
  - 第一层（外层）：`8s`，缓慢旋转
  - 第二层（中层）：`4s`，中等速度
  - 第三层（内层）：`2s`，快速旋转
- 速度倍数关系（1:2:4）确保每 8s 一个完整周期对齐，形成可预测的节奏变化
- 不同速度的叠加产生视觉上的“追逐”效果，层次丰富且动感十足

### 性能考虑
- 动画仅依赖 `transform: rotateY()`，由 GPU 合成层驱动，避免重排与重绘
- `perspective` 属性在现代浏览器中硬件加速支持良好，性能开销低
- 三层元素均为绝对定位，不触发文档流变化，渲染性能稳定

### 适用场景
- 需要层次分明、科技感的加载指示
- 3D 风格应用或数据可视化场景的忙碌反馈
- 强调“处理中”状态的多层次动画展示

总体而言，该效果以极简的三层嵌套结构与纯 CSS 实现了层次清晰的 3D 旋转动画，速度差异化带来的追逐感与高光装饰的点缀，共同营造出具有深度与节奏的视觉体验。
