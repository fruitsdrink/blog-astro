# 代码解析

### HTML 结构

使用了4个`<span>`元素作为蛇形边框的四个边，包裹在一个`.square`容器中。

### CSS 核心实现

#### 1. 容器旋转 (`.square`)

- **`animation: rotate 8s linear infinite`**: 整个正方形以8秒为周期持续逆时针旋转
- **`border: 10px dashed rgba(0, 0, 0, 0.2)`**: 设置虚边框作为引导线
- **`box-shadow`**: 使用双重阴影创建边框视觉效果

#### 2. 边框动画 (`@keyframes animate`)

- **0%**: 边框从左侧完全移出视野(`left: -100%`)
- **25%**: 边框移入视野(`left: 0`)
- **50%-100%**: 边框移动到右侧消失(`left: 100%`)

#### 3. 四个边的定位

- 使用`transform: rotate()`将四个`<span>`分别旋转0°、90°、180°、270°
- 每个`<span>`的`::before`伪元素通过`border-top`创建边框线条
- 设置不同的`animation-delay`(0s、0.5s、1s、1.5s)实现蛇形移动的连贯效果

#### 4. 背景和布局

- 使用`linear-gradient`创建渐变背景
- 通过`display: flex`实现垂直居中布局

这个加载器的核心是四个边框线条按顺序出现和消失，配合容器的旋转，形成蛇形追逐的视觉效果。
