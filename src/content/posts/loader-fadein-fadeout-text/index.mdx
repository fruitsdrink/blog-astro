## 淡入淡出文本加载器效果分析

这是一个 3D 翻转文字加载动画。每个字母依次从侧后方旋转进入视野，逐渐放大并淡入，随后缩小淡出，形成流畅的波浪式视觉节奏。

### 结构与布局

- 容器使用 `<ul>` 列表，每个字母为独立的 `<li>` 元素
- 每个 `<li>` 通过 `display: table-cell` 实现水平排列，自动间距对齐
- 使用 CSS 变量 `--i`（0-6）为每个字母分配序号，用于错峰延迟计算
- `transform-origin: left` 设置旋转原点为左侧，让翻转动画从字母左侧开始

### 视觉设计

- 字体设置为 `5em`、白色、加粗，确保醒目突出
- 背景使用深蓝色 `#004eff`，与白色文字形成高对比度
- 字母采用纯色设计，简洁现代，适合加载场景

### 动画与节奏

- 核心动画 `@keyframes animate` 分为三个阶段：
  - **0%**：完全透明 `opacity: 0`，绕 Y 轴旋转 90 度（侧向），X 轴平移 -30px（从后侧进入）
  - **50%**：完全不透明 `opacity: 1`，Y 轴旋转到 0 度（正面），X 轴平移 30px，缩放至 0.4 倍（突出焦点）
  - **100%**：再次变透明 `opacity: 0`，保持正面朝向
- 动画时长 3s，线性无限循环
- 通过 `animation-delay: calc(var(--i) * 0.2s)` 实现阶梯式延迟（0s、0.2s、0.4s...），让相邻字母错峰启动
- 这种延迟机制形成“波浪式”推进效果，视觉上更具流动感与层次

### 性能考虑

- 动画主要依赖 `transform`（rotateY、translateX、scale）与 `opacity`，均由合成层驱动，性能优秀
- 避免使用会引起重排的属性（如 width、height、left、top）
- 线性动画函数（linear）计算开销低，适合无限循环

### 适用场景

- 需要吸引注意力的加载与过渡界面
- 品牌展示或欢迎页面的文字动画
- 需要科技感与动态感的文字效果

整体而言，该效果以纯 CSS 与极简 DOM 结构实现了富有节奏感的 3D 翻转文字动画，每个字母的依次登场与淡出形成流畅的视觉叙事，适合现代 Web 应用的加载反馈。
