## 旋转圆形加载器效果分析

这是一个由 8 个半透明圆形组成的扇形旋转加载动画。通过围绕底部中心点的旋转与阶梯式延迟启动，形成富有节奏感的扇形展开效果。

### 结构与布局

- 容器 `<ul>` 固定尺寸为 200×200，作为所有圆形元素的定位参照
- 8 个 `<li>` 元素均使用 `position: absolute`，完全重叠于同一位置（`top: 0`）
- 每个圆形继承容器的 100% 宽高，通过 `border-radius: 50%` 形成完整圆形
- 所有圆形共享相同的尺寸，仅通过颜色与启动时间区分

### 视觉设计与色彩

- 8 个圆形采用从浅到深的紫色渐变：从 `rgba(214,188,211,0.8)` 过渡到 `rgba(48,83,144,0.4)`
- 半透明背景（alpha 值 0.4-0.8）叠加产生丰富的层次与混合效果
- `box-shadow: 0 0 5px rgba(0, 0, 0, 0.5)` 为每个圆形增加轻微投影，增强立体感

### 动画机制与旋转原点

- 核心动画 `@keyframes animate`：0°→360° 的连续旋转，时长 7s，线性无限循环
- **关键特性**：`transform-origin: bottom center` 将旋转中心点设置在圆形的底部中心
- 旋转原点改变使得圆形围绕底部展开，形成扇形扇出的视觉，而非普通的中心旋转
- 通过 CSS 变量 `--i`（值为 1-8）与 `animation-delay: calc(var(--i) * 1s)` 实现阶梯式延迟
- 相邻圆形依次延迟 1 秒启动，形成波浪式的扇形展开序列

### 节奏与动感

- 7 秒的总时长配合 8 个圆形、1 秒间隔的延迟，创造流畅的连续旋转节奏
- 延迟启动让圆形形成"追逐"效果，从第一个圆形开始，依次展开形成完整的扇形
- 由于圆形完全重叠且半透明，叠加区域会产生更深的颜色混合，形成视觉重心

### 性能考虑

- 动画仅依赖 `transform` 属性，由 GPU 合成层驱动，性能优异
- 使用 CSS 变量实现延迟计算，代码简洁且易于维护
- 无布局回流与重绘，适合长时间运行的加载场景

### 适用场景

- 需要柔和、优雅视觉风格的加载指示
- 适合紫色或渐变色彩主题的应用
- 扇形展开效果适合强调"逐步展开"或"逐步加载"的语义

总体而言，该效果以极简的 DOM 结构与纯 CSS 实现了富有节奏感的扇形旋转动画，旋转原点的巧妙设置与延迟启动机制共同营造出优雅的视觉层次。
